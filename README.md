# CodeBot
This is a Twitter chatbot that respond to every tweet he's tagged in by running the text in python.

## Live Bot
This bot *is* live right now, although it's still in the testing stage, here's a link:
https://twitter.com/CodeBot5
Please feel free to test him :)

## Example
If you tag him in a tweet saying: `4*sum((-1.0)**(n%2) / (2*n + 1) for n in range(2010))` he'll reply with: `3.1410951411827663`

Here is the example working: https://twitter.com/CodeBot5/status/1041475137105784833
